# FitBit API


## Environment values

Set following values which can be retrieved from [dev.fitbit.com](https://dev.fitbit.com/):

- **FITBIT_CLIENT_ID** : client_id

- **FITBIT_CLIENT_SECRET** : client_secret

- **FITBIT_REDIRECT_URI** : redirect_uri


## How to call API

- Get access token:

  - `GET /auth/fitbit`

- Call API:

  - (For example, )`GET /profile`
